# RoofMaps

A landing page for a hail tracking platform with an interactive map.

## Files

- `index.html` - Main page structure
- `styles.css` - Styling
- `script.js` - Map and interactions

## Setup

1. Clone or download the files
2. Open `index.html` in your browser

That's it! No installation needed.

## Tech Used

- HTML/CSS/JavaScript
- MapLibre GL JS for the map
- Google Fonts (Roboto)

## Customize

- **Colors**: Edit `styles.css` (primary: `#1e3a8a`, accent: `#facc15`)
- **Map location**: Change `center: [-98, 39]` in `script.js`
- **Hero image**: Update the background URL in `styles.css`
